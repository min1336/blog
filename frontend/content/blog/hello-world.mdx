# 안녕하세요!

첫 번째 블로그 포스트입니다. 이 블로그는 Next.js 16과 MDX를 사용하여 만들었습니다.

## 기술 스택

이 블로그에서 사용하는 기술 스택입니다:

- **Next.js 16** - React 프레임워크
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 유틸리티 CSS
- **MDX** - 마크다운 + JSX

## 코드 예시

타입스크립트 코드 하이라이팅 예시입니다:

````typescript
interface BlogPost {
  title: string
  date: string
  tags: string[]
}

const post: BlogPost = {
  title: "첫 번째 포스트",
  date: "2026-03-24",
  tags: ["blog", "intro"],
}

console.log(post.title)
````
